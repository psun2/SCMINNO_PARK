만약 github 다른 아이디를 쓰고 싶으면, 어떻할까요?

​

window 같은 경우에는 처음 한번만 로그인 하면, 로그인 기록이 자동 저장되어, 아이디 비밀번호를 묻지 않습니다.

지금부터 그 자동로그인을 한 아이디를 로그아웃 하는 작업을 해보도록 하겠습니다.
